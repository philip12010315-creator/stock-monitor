# 籌碼觀察系統 (Stock Monitor)

本專案是一個基於 Node.js 與 Firebase 的外資成本監控儀表板。

## 🚀 快速查看
- **網址**: [https://philip12010315-creator.github.io/stock-monitor/](https://philip12010315-creator.github.io/stock-monitor/)
- **功能**: 即時監控 XQ 匯出的外資成本資料，並提供跨裝置（手機/電腦）的收藏同步功能。

## 💻 換台電腦如何繼續？

如果您在公司或新環境想查看進度：
1. **直接開啟網頁**：直接點擊上面的網址，輸入您的 ID 與密碼登入。
2. **查看專案內容**：在 GitHub 頁面查看 `index.html` 與 `server.js`。

### 若要在公司電腦「更新資料」：
1. **需安裝 Node.js**。
2. **下載專案**: `git clone [專案網址]`。
3. **執行伺服器**: `node server.js`。
4. **放入新 CSV**: 將 XQ 匯出的檔案放入 `發動結果` 資料夾。
5. **同步雲端**: 執行 `同步到GitHub.bat`。
