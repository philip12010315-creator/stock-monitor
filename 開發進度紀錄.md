## 🚀 已完成功能與優化

### 1. 座標系統精確對齊 (Bug Fix)
- **問題**：原本「獲利路徑推演」的進度條與上方價格標籤（如 $142.5）位置對不上。
- **修復**：重新設計了節點佈局 CSS，將 `.path-node-col` 設為 `width: 0` 並配合 `justify-content: space-between`，確保節點中心點精準落在 0%, 25%, 50%, 75%, 100% 的位置。

### 2. 資料日期自動解析 (免改 XQ 腳本)
- **邏輯**：使用者要求不准改動 XQ 腳本。
- **實現**：修改 `server.js`，現在會自動讀取匯出 CSV 的 **第二行 (Line 2)**，使用正規表達式抓取「資料日期：XXXX年XX月XX日」字串，並同步顯示於網頁頂端。

### 3. 全新跑馬燈 (Status Ticker)
- **視覺**：頂部資訊（日期、大盤、總數）改為專業股市跑馬燈效果。
- **無縫循環**：採用「30倍內容重複」+「3.33% 位移重置」技術，確保在手機與電腦上皆能無縫、無限循環滾動。
- **速度設定**：
    - 電腦版：18秒 / 循環。
    - 手機版：12秒 / 循環（獨立加速）。
- **佈局**：手機版自動切換為「滿版貼邊」設計，移除圓角。

### 4. 手機版收藏功能 (UX 優化)
- **按鈕**：右下角新增懸浮星星按鈕。
- **視窗**：點擊後滑出半透明 Bottom Sheet (側邊抽屜)，方便手機用戶查看收藏清單。

### 5. 視覺美化 (Theme Polish)
- **淺色模式**：修復了白色背景下文字不清晰、面板變灰的問題，優化了對比度。
- **大盤顯示**：網頁頂部同步顯示加權指數價格。

## 🛠️ 技術環境說明

- **後端 (Node.js)**：`server.js` 負責監控 `發動結果` 資料夾。
- **數據交換**：透過 WebSocket (WS) 即時推送 JSON。
- **資料儲存**：每次更新會同步產生一份 `data.json`，支援 GitHub Pages 靜態讀取。
- **路徑注意**：專案路徑目前位於 `Desktop\測試`，請確保新電腦上的資料夾結構一致。
